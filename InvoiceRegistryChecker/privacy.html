Privacy Policy

プライバシーポリシー Privacy Policy

「登録番号検索くん」（以下「本アプリ」）は、利用者のプライバシーを尊重し、アプリの機能提供に必要な範囲で情報を取り扱います。 本ポリシーでは、本アプリにおける情報の利用目的と取り扱い方法を説明します。 "Invoice Registry Checker" (the "App") respects user privacy and handles information only as necessary to provide the App's features. This policy explains what information is used and how it is handled.

最終更新日 / Last Updated: 2026年5月16日 / May 16, 2026

1. 取得または利用する情報 / Information We Access or Use

本アプリは、利用者が許可した場合に限り、端末のカメラおよび写真ライブラリを利用します。 カメラまたは選択された画像は、請求書や領収書などに記載された適格請求書発行事業者の登録番号を読み取るために使用されます。

The App uses the device camera and photo library only when the user grants permission. Camera captures or selected images are used to read invoice registration numbers shown on invoices, receipts, and similar documents.

本アプリは、画像から読み取ったテキスト、登録番号候補、検索した登録番号、検索結果、検索履歴を取り扱う場合があります。

The App may handle recognized text, registration number candidates, searched registration numbers, lookup results, and search history.

2. 利用目的 / Purpose of Use

本アプリで扱う情報は、以下の目的のために利用します。

Information handled by the App is used for the following purposes:

  • 画像内の登録番号を端末上のOCRで読み取るため / To read registration numbers in images using on-device OCR
  • 登録番号を国税庁の適格請求書発行事業者公表システムAPI、または開発者が管理する中継サーバー経由で検索するため / To look up registration numbers through the National Tax Agency API or a developer-managed relay server
  • 検索結果をアプリ内に表示するため / To display lookup results in the App
  • 検索履歴を端末内に保存し、後から確認できるようにするため / To store search history on the device for later review
  • OCRの読み取り結果を確認し、機能改善や不具合調査を行うため / To review OCR results for feature improvement and troubleshooting

3. 端末内での保存 / On-Device Storage

検索履歴は、利用者の端末内に保存されます。履歴には、検索した登録番号、検索結果、検索日時が含まれる場合があります。 利用者が「読み取り画像を履歴に保存」を有効にした場合、読み取りに使用した画像とOCR結果も端末内に保存される場合があります。

Search history is stored on the user's device. It may include searched registration numbers, lookup results, and search dates. If the user enables saving scanned images to history, the image used for OCR and OCR result may also be stored on the device.

4. サーバーへの送信 / Server Transmission

登録番号を検索するため、本アプリは検索対象の登録番号を国税庁のAPI、または開発者が管理する中継サーバーへ送信します。 中継サーバーを利用する場合、中継サーバーは登録番号を国税庁APIへ送信し、検索結果を本アプリへ返します。

To perform lookups, the App sends the target registration number to the National Tax Agency API or to a developer-managed relay server. When the relay server is used, it forwards the registration number to the National Tax Agency API and returns the lookup result to the App.

OCRで読み取った画像を使って検索した場合、読み取りに使用した画像、OCRで認識したテキスト、登録番号候補、検索結果が、機能改善や不具合調査のために開発者が管理するサーバーへ送信される場合があります。

When a lookup is performed from an OCR image, the image used for OCR, recognized text, registration number candidate, and lookup result may be sent to a developer-managed server for feature improvement and troubleshooting.

5. 第三者提供 / Third-Party Sharing

本アプリは、利用者の情報を広告目的で第三者へ販売、共有、提供しません。 登録番号検索のために、検索対象の登録番号は国税庁APIへ送信されます。

The App does not sell, share, or provide user information to third parties for advertising purposes. Registration numbers are sent to the National Tax Agency API for lookup purposes.

6. 解析ツール・広告SDK / Analytics and Advertising SDKs

本アプリは、利用者の行動追跡を目的とした広告SDKや、トラッキング目的の外部解析サービスを使用しません。

The App does not use advertising SDKs for behavioral tracking or external analytics services for tracking purposes.

7. 権限の管理 / Permission Control

カメラおよび写真ライブラリへのアクセス権限は、iPhoneの設定画面からいつでも変更または取り消すことができます。 権限を許可しない場合、撮影や画像選択を使った読み取り機能は利用できませんが、手入力による検索は利用できます。

Camera and photo library permissions can be changed or revoked at any time in iPhone Settings. If permission is not granted, camera capture or image selection features will not be available, but manual lookup can still be used.

8. 子どもの個人情報 / Children's Privacy

本アプリは、意図的に子どもの個人情報を収集することを目的としていません。

The App is not intended to intentionally collect children's personal information.

9. ポリシーの変更 / Changes to This Policy

本ポリシーは、法令やアプリ仕様の変更に応じて改定される場合があります。重要な変更がある場合は、本ページ上で告知します。

This policy may be updated in response to legal requirements or changes to the App. Material changes will be announced on this page.
