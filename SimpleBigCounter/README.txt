GitHub Pages の公開設定:
1. GitHub でこのリポジトリを開く
2. Settings > Pages に移動する
3. Build and deployment で以下を選択する:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
4. Save を押す
5. 1〜2分後に以下のURLで公開される:
   https://hotaru2122.github.io/SimpleBigCounterSite/
